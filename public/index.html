Scott's Simple Weather API

Quickly deploy an Express.js API to serve up weather data for a location you define. Your server caches the weather data for 5 minutes so you don't exceed the One Call API's free tier limit of 1,000 requests per day.

Quickstart

  1. 1.
    Generate an API key for OpenWeather's One Call API v3.0.
  2. 2.
    Clone this repo.
  3. 3.

    Get the latitude & longitude coordinates for your location from Google Maps.

    • • Right-click on a place on the map
    • • Click on the top item XX.xxx, XX.xxx to copy it to clipboard
  4. 4.

    Add your API key, plus the coordinates, a location name, and an optional port to an .env file: 

    # required
LATITUDE=38.8977
LONGITUDE=-77.0365
LOCATION_LABEL="Washington, DC" # any text string

OPENWEATHER_KEY=xxxxxxxxxxxxxxxxxxxxxxxxxxxxxx # your OpenWeather API key
API_KEY=xxxxxxxxxxxxxxxxxxxxxxxxxxxx # key other computers will use to access this server

# optional
PORT=4000
NODE_ENV=development # enables localhost:4000 CORS

    Note: npm start creates an API_KEY in .env if one is not detected.

  5. 5.
    Deploy your server to a VPS or cloud and start building notifications and adding data and automations to your other apps and websites.

Authentication Required

All API endpoints require an API key. Include the X-API-Key header in your requests.

X-API-Key: your-api-key-here

How to Use

Simplified data:

curl -H "X-API-Key: your-api-key" http://your-server:4000/api/

Full data:

curl -H "X-API-Key: your-api-key" http://your-server:4000/api/full

Simplified data:

async function getWeather() {
  try {
    const response = await fetch('http://your-server:4000/api/', {
      headers: {
        'X-API-Key': 'your-api-key'
      }
    });
    const data = await response.json();
    console.log(data);
  } catch (err) {
    console.error(err);
  }
}

Full data:

async function getFullWeather() {
  try {
    const response = await fetch('http://your-server:4000/api/full', {
      headers: {
        'X-API-Key': 'your-api-key'
      }
    });
    const data = await response.json();
    console.log(data);
  } catch (err) {
    console.error(err);
  }
}

Simplified data:

import requests

headers = {'X-API-Key': 'your-api-key'}
response = requests.get(
    'http://your-server:4000/api/', 
    headers=headers
)
data = response.json()
print(data)

Full data:

headers = {'X-API-Key': 'your-api-key'}

API Endpoints

Returns simplified weather data with essential fields only.
